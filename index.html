mungo

Mungo is set of bash scripts source which help you deploy your apps like heroku on amazon ec2 linux ami using foreman.

By default your app will start on port 5000

install

sudo -s

yum update

yum remove ruby

curl -L get.rvm.io |bash -s stable

rvm install 1.9.3

ruby -v

gem install foreman

git clone git://github.com/hackable/mungo.git && cd mungo

chmod +x install.sh && ./install.sh

If you want to run coffee-script based apps u need to install coffee-script with

npm install -g coffee-script

local repo to remote deploy

git init

git add .

git commit -a -m "My first commit"

git remote add cloud ec2-user@EC2_PUBLIC_URL:/home/ec2-user/repos/PROJECT_NAME

git push cloud master

commands

  • ./mungo.sh --create [project_name]

  • ./mungo.sh --destory [project_name]

  • ./mungo.sh --version

deployment

create the following file in the root of your app if you want to execute pre and post deploy commands:-

  • mungo_pre_deploy - put all your pre deploy commands here such as npm install for node apps and bundle

  • mungo_post_deploy - put all post deploy commands
